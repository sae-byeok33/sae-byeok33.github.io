# BOOTING UP
우리가 컴퓨터를 실행시키기 위해서 파워버튼을 누르면 무슨일이 일어날까?  

## BASIC STEP
1. BIOS 시작
2. CMOS에서 설정 불러오기
3. 장치 초기화
4. POST(Power on self test)실행
5. 부트스트랩 시퀀스 시작

### 1.Starting the BIOS
+ **기본 입출력 시스템(BIOS)**  
    → 칩에 내장된 미니 OS  
+ **PC 전원이 켜지면 즉시 실행이 시작된다.**  
    → BIOS 칩의 코드가 RAM의 낮은 주소(ex:0xFF)에 복사된다.  
    → jmp 0xff(16bits 명령어)가 0xFFFF0 (2^20 -16)에 기록된다.
